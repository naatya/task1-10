 Cyber Security Basics &amp; Attack Surface
**8.Summary: Putting It All Together**
Cybersecurity is the practice of protecting the CIA Triad across various attack surfaces. Whether it's a Script Kiddie trying to deface a website or a Nation-State trying to steal secrets, they all look for gaps in the OWASP Top 10. By mapping the data flow from a user's phone to a central database, we can identify "choke points" where encryption, firewalls, and strict access controls must be placed to keep data safe.
